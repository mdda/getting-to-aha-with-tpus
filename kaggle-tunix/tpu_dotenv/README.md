## .env credential setting : `dotenv`

* Kaggle tokens:
  + from https://www.kaggle.com/settings : API Tokens (Recommended)


`dotenv` File format : 

```bash
KAGGLE_USERNAME="YOUR_KAGGLE_USERNAME"
KAGGLE_KEY="YOUR_KAGGLE_API_KEY"
# etc...
```

The file `dotenv` here will be stored as `/home/tpu_user/.env` on the TPU instance
